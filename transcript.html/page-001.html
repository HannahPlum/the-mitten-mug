# Claude Code Transcripts Tool

## Overview
This article describes **claude-code-transcripts**, a Python CLI tool that converts Claude Code sessions into detailed, shareable HTML pages that provide better visibility into coding work than the native Claude Code interface.

## Key Features
The tool generates organized HTML transcripts including:
- Summary pages with initial prompts and overview
- Multi-page detailed breakdowns of all activities
- Integration with Claude's thinking traces (usually hidden)
- Support for both local Claude Code and Claude Code for web sessions

## Installation
The quickest method requires `uv` installed:
```
uvx claude-code-transcripts
```

Alternative installation approaches include:
- `uv tool install claude-code-transcripts`
- `pip install claude-code-transcripts`

## Usage
After running the tool, users can:
1. Select from a list of local Claude Code sessions
2. Generate HTML transcripts automatically
3. Optionally publish to GitHub Gists using the `--gist` flag
4. Access web-based sessions with: `uvx claude-code-transcripts web --gist`

## Technical Details
Simon Willison built the entire tool using Claude itself. It uses dependencies like Click, Jinja2, HTTPx, and Markdown, and includes reverse-engineered API access for "Claude Code for web" sessions.
